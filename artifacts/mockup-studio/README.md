# Mockup Studio Pro 🚀

**Mockup Studio Pro** es una plataforma de creación de mockups 3D de alta fidelidad, diseñada para ofrecer una experiencia premium tanto en escritorio como en dispositivos móviles. Nuestro objetivo es democratizar el diseño 3D profesional, llevando la potencia de aplicaciones nativas como Rotato directamente al navegador y al bolsillo del usuario.

## 🌟 Nuestra Ventaja Competitiva: Omnicanalidad
A diferencia de otras herramientas que se limitan al escritorio, **Mockup Studio Pro** ha sido construida bajo una arquitectura *Cross-Device Full-Power*. Todo lo que puedes hacer en una PC de 27 pulgadas, lo puedes hacer con la misma fluidez y precisión en la palma de tu mano.

## 🗺️ Roadmap: El Camino a la Excelencia (Meta: Rotato Killer)

Este listado representa nuestras metas a corto y mediano plazo para elevar la app al siguiente nivel de competición profesional.

### 🎨 Fase 1: Realismo Extremo y Materiales
*Meta: Lograr que el renderizado sea indistinguible de una fotografía real.*
- [x] **Overhaul de Materiales PBR:** Implementar mapas de rugosidad (roughness) y metalicidad avanzados para el iPhone 16 y otros modelos.
- [x] **Integración de HDRI Studio:** Biblioteca de entornos de iluminación profesional (Softbox, Minimalist, Outdoor) para reflejos realistas.
- [x] **Sombras de Contacto (SSAO):** Implementar oclusión ambiental para que los dispositivos se sientan "pesados" y bien apoyados en el suelo.


### 🎬 Fase 2: Animación Cinematográfica
*Meta: Que cualquier usuario pueda crear trailers de productos con un clic.*
- [ ] **Curvas Bezier en la Timeline:** Control granular de la aceleración (easings) para movimientos de cámara fluidos.
- [ ] **Biblioteca de "Camera Actions":** Presets de un solo clic como "Hero Spin", "Dramatic Tilt" o "Macro Zoom".
- [ ] **Modo Audio-Reactivo:** Sincronización opcional de la luz y la cámara con la frecuencia de la música de fondo.

### 🛠️ Fase 3: Flujo de Trabajo e Interactividad
*Meta: Integrarse perfectamente en el ecosistema del diseñador moderno.*
- [ ] **Live Figma / Prototyping Sync:** Soporte para IFrames e integraciones directas para navegar por la pantalla del dispositivo en 3D.
- [ ] **IA Scene Architect:** Generación de estilos (luces, fondos y colores) mediante prompts de texto.
- [ ] **Escenarios 3D (Stages):** Añadir pedestales, mesas y entornos 3D reales, no solo fondos planos.

### 📦 Fase 4: Exportación de Grado Profesional
*Meta: Entregar archivos listos para post-producción de alto nivel.*
- [ ] **Video con Transparencia (Alpha Channel):** Exportación en WebM/MOV con canal alfa para editores de video.
- [ ] **Renderizador Frame-by-Frame:** Garantizar 60fps perfectos en la exportación, independientemente de la potencia del dispositivo del usuario.
- [ ] **Captura 4K:** Motor de capturas de pantalla de ultra alta resolución para material impreso.

### 📱 Fase 5: Expansión de Ecosistema
- [ ] **Orquestación Multi-Dispositivo:** Poder añadir y animar una familia de productos (iPhone + Mac + Watch) en la misma escena.

## 🛠️ Stack Tecnológico
- **Core:** React + TypeScript
- **3D Engine:** Three.js (@react-three/fiber)
- **Efectos:** Post-processing (@react-three/postprocessing)
- **Animación:** Framer Motion + Timeline Propia
- **UI:** CSS Custom Tokens (Glassmorphism)
